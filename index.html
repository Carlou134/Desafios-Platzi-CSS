Descripción del Proyecto

Este proyecto forma parte de una serie de desafíos propuestos en Platzi, orientados a reforzar mis conocimientos en HTML y CSS sin utilizar aún técnicas de diseño responsivo como Flexbox o Media Queries. El objetivo principal ha sido poner en práctica la maquetación clásica con float, gestión del flujo del documento, y estructura semántica de páginas web.

Durante el desarrollo:

  • Implementé una barra de navegación fija con elementos alineados a la izquierda y derecha usando float.
  • Construí un layout de dos columnas en el cuerpo principal, posicionando elementos visuales al lado izquierdo y contenido textual a la derecha.
  • Aplicando solo propiedades básicas de CSS, logré dividir la página de manera clara, simulando la estructura de una plataforma educativa.
  • Me enfoqué en reforzar habilidades como el uso de box-model, clases, paddings, bordes y el posicionamiento tradicional sin apoyo de Flexbox.



Esta etapa es fundamental antes de introducirme en diseño responsivo, ya que me permite comprender cómo se comportan los elementos y cómo gestionar el espacio en pantalla solo con CSS clásico.
